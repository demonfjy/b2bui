单选 radio

单选 radio
项目结构
{{#radio1}} {{> radio/radio}} {{/radio1}} {{#radio2}} {{> radio/radio}} {{/radio2}}
组件结构
<label class="bl-radio-wrapper {{class}}">
    <span class="bl-radio ">
        <input type="radio" id="{{id}}" name="{{name}}" class="bl-radio-input" {{#if disabled}}  disabled {{/if}} value="on" {{checked}}>
        <span class="bl-radio-inner"></span>
    </span>
    <span>{{{text}}}</span>
</label>

{{class}}此处为json数据，可通过json自行定义class。例如，我需要分页向右移动10像素，可通过此处类名修改。{{id}}定义按钮id。{{#if disabled}}判断时候可点击。

组件样式调用（sass文件或css文件）
@import "tobe/components/radio/radio";

这是组件所需调用的组件样式。

组件结构调用
{{#radio1}}
    {{> radio/radio}}
{{/radio1}}
{{#radio2}}
    {{> radio/radio}}
{{/radio2}}

{{#radio1}}、{{#radio2}}为json数据名称，可自行定义，{{> radio/radio}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/radio.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/radio.js"
    },
    "radio1": {
        "class": "",
        "id": "",
        "name": "mc1",
        "text": "11"
    },
    "radio2": {
        "class": "",
        "id": "",
        "name": "mc1",
        "text": "22"
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“radio1”、“radio2”组件调用数组名称一致。

js调用
require(['radiot/radio'], function() {});

此处调用的是组件效果的js，如果不调用组件js不会产生效果。
