{{class}}此处为json数据，可通过json自行定义class。例如，我需要分页向右移动10像素，可通过此处类名修改。{{id}}定义按钮id。{{#if disabled}}判断时候可点击。

组件样式调用（sass文件或css文件）
@import "tobe/components/show/show";

这是组件所需调用的组件样式。

组件结构调用
{{#show1}}
    {{> show/show}}
{{/show1}}

{{#show1}}为json数据名称，可自行定义，{{> show/show}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/show.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/show.js"
    },
    "show1": {
        "infoshow":"1",
        "class":"show-main",
        "style":"w415",
        "name":"确认要取消申请吗？",
        "text":"确定取消后该申请将会立即作废。",
        "button":[{
            "class":"bl-btn-lg",
            "id":"cancel",
            "name":"取消"
        },
        {
            "class":"bl-btn-primary bl-btn-lg",
            "id":"",
            "name":"确定"
        }]
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“radio1”、“radio2”组件调用数组名称一致。

js调用
require(['show/show'], function() {});

此处调用的是组件效果的js，如果不调用组件js不会产生效果。
