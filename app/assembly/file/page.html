分页 page

page 分页，分页目前只提供一种形式。用户可通过class自行调整。
普通分页
{{#page1}} {{> page/page}} {{/page1}} 
组件结构
<ul class="bl-pagination {{class}}" unselectable="unselectable">
    <li class="bl-pagination-disabled bl-pagination-prev"><a></a></li>
    <li title="1" class="bl-pagination-item bl-pagination-item-active"><a>1</a></li>
    <li title="2" class="bl-pagination-item"><a>2</a></li>
    <li title="3" class="bl-pagination-item"><a>3</a></li>
    <li title="4" class="bl-pagination-item"><a>4</a></li>
    <li title="5" class="bl-pagination-item"><a>5</a></li>
    <li class="bl-pagination-next"><a></a></li>
</ul>

{{class}}此处为json数据，可通过json自行定义class。例如，我需要分页向右移动10像素，可通过此处类名修改。

组件样式调用（sass文件或css文件）
@import "icon";//图标样式
@import "tobe/components/page/page";

这是组件所需调用的组件样式。

组件结构调用
{{#page1}}
    {{> page/page}}
{{/page1}}

{{#page1}}为json数据名称，可自行定义，{{> page/page}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/page.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/page.js"
    },
    "page1" : {//与组件对应的数据名称
        "class" : ""
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“page1”与组件调用数组名称一致。可定义“class”更改page自己样式
