表格 table

table 表格
项目结构
{{#table1}} {{> table/table}} {{/table1}}
组件结构
<div class="bl-table {{style}}">
    <table width="100%" border="0" cellspacing="0" cellpadding="0">
        <tr>
            {{#each th}}
                <th width="{{width}}" align="{{align}}">{{{text}}}</th>
            {{/each}}
        </tr>
        {{#each tr}}
            <tr>
                {{#each td}}
                    <td align="{{align}}">{{{text}}}</td>
                {{/each}}
            </tr>
        {{/each}}
    </table>
</div>

{{style}}此处为json数据，可通过json自行定义class。例如，我需要分页向右移动10像素，可通过此处类名修改。{{width}}为宽度，{{align}}为提示，{{{text}}}为字符串数据。{{#each th}}遍历循环标题，{{#each tr}}遍历循环表格行数。

组件样式调用（sass文件或css文件）
@import "tobe/components/table/table";

这是组件所需调用的组件样式。

组件结构调用
{{#table1}}
    {{> table/table}}
{{/table1}}

{{#table1}}为json数据名称，可自行定义，{{> table/table}}为组件在项目中的位置。
