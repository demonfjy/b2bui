多行文本输入框 textarea

textarea 多行文本输入框
项目结构
{{#textarea1}} {{> textarea/textarea}} {{/textarea1}}
组件结构
<textarea type="textarea" rows="{{height}}" class="bl-input {{class}}" value="" id="{{id}}" {{#if disabled}} disabled {{/if}} placeholder="{{placeholder}}"></textarea>
组件样式调用（sass文件或css文件）
@import "tobe/components/input/input";

这是组件所需调用的组件样式。

组件结构调用
{{#textarea1}}
    {{> textarea/textarea}}
{{/textarea1}}

{{#textarea1}}为json数据名称，可自行定义，{{> textarea/textarea}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/textarea.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/textarea.js"
    },
    "textarea1" : {
        "height" : "4",
        "class" : "",
        "id" : "",
        "placeholder" : "输入备注"
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“textarea”
