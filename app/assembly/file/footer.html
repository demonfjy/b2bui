{{#each gl.scripts}} 变量主要用于存放js库，例如：Dojo 、Scriptaculous 、Prototype 、yui-ext 、Jquery 、Mochikit、mootools 、moo.fx。{{#if footerbottom}}判断选择哪种尾部。

组件结构调用
{{#footer1}}
    {{> footer/footer}}
{{/footer1}}
{{#footer2}}
    {{> footer/footer}}
{{/footer2}}

{{#footer1}}、{{#footer2}}为json数据名称，可自行定义。{{> footer/footer}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/footer.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/footer.js"
    },
    "footer1":{
    },
    "footer2":{
        "footerbottom":"1"
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“footer1”、"footer2"与组件调用数组名称一致。“footerbottom”：“1”，判断使用哪种尾部。
