复选框 checkbox

checkbox复选框组件，此处组件提供了2种复选框样式和用法。其中一种是可选复选框，一种是不可选复选框。
项目结构
{{#checkbox1}} {{> checkbox/checkbox}} {{/checkbox1}} {{#checkbox2}} {{> checkbox/checkbox}} {{/checkbox2}} 
组件结构
<label class="bl-checkbox-wrapper {{class}}">
    <span class="bl-checkbox ">
        <input type="checkbox" class="bl-checkbox-input" {{#if disabled}} disabled {{/if}} value="on" name="{{name}}" id="{{id}}" {{checked}}>
        <span class="bl-checkbox-inner"></span>
    </span>
    <span>{{text}}</span>
</label>

{{class}}此处为json数据，可通过json自行定义class。例如，我需要分页向右移动10像素，可通过此处类名修改。{{#if disabled}}判断是否为可选。{{name}}此处为组件名称，注意当需要多个组件组成一个整体时，此处名称需要一致进行关联。{{text}}为内容。

组件样式调用（sass文件或css文件）
@import "tobe/components/checkbox/checkbox";

这是组件所需调用的组件样式。

组件结构调用
{{#checkbox1}}
    {{> checkbox/checkbox}}
{{/checkbox1}}	
{{#checkbox2}}
    {{> checkbox/checkbox}}
{{/checkbox2}}

{{#checkbox1}}、{{#checkbox1}}为json数据名称，可自行定义，{{> checkbox/checkbox}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/checkbox.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/checkbox.js"
    },
    "checkbox1":{
        "text": "30天内免登录",
        "class":"",
        "other":"<a href=''>忘记密码</a>"
    }, 
    "checkbox2":{
        "text": "30天内免登录",
        "class":"bl-checkbox-disabled",
        "disabled":"1"
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“checkbox1”、“checkbox2”与组件调用数组名称一致。此处组件分为可选和不可选两种，添加"class":"bl-checkbox-disabled"为不可选，不添加为默认可选。

js调用
require(['checkbox/checkbox'], function() {});

此处调用的是组件效果的js，如果不调用组件js效果不会产生。['checkbox/checkbox']此处可以调用多个组件js或者自己定义的js。function() {}中自定义js。其中['checkbox/checkbox']中的文件位置是相对于html的，而不是这个项目的。
