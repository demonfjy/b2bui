组件结构调用
{{#transfer1}}
    {{> transfer/transfer}}
{{/transfer1}}

{{#transfer1}}为json数据名称，可自行定义，{{> transfer/transfer}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/transfer.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/transfer.js"
    },
    "transfer1" : {
        "checkbox1" :{
            "class" :"",
            "name":"",
            "id":"",
            "text":"11"
        },
        "name1":"1",
        "input":{
            "type":"text",
            "id":"",
            "name":"",
            "class":"",
            "placeholder":"1"
        },
        "check":{
            "class" :"",
            "name":"",
            "id":"",
            "text":"11"
        },
        "check1":{
            "class" :"",
            "name":"",
            "id":"",
            "text":"11"
        },
        "checkbox2":{
            "class" :"",
            "name":"",
            "id":"",
            "text":"11"
        },
        "name2":"1"
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“transfer1”、“checkbox1”、“name1”、“input”、“check”、“check1”、“checkbox2”、“name2”组件调用数组名称一致。

js调用
require(['transfer/transfer'], function() {});

此处调用的是组件效果的js，如果不调用组件js不会产生效果。
