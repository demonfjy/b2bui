开始使用 - 入门

项目运行环境说明，项目结构详解。
组件下载：点击下载
运行环境

1、安装nodejs

1.1、说明：gulp是基于nodejs，理所当然需要安装nodejs；
1.2、安装：打开nodejs官网，点击硕大的绿色Download按钮，它会根据系统信息选择对应版本（.msi文件）。然后像安装QQ一样安装它就可以了（安装路径随意）。

2.安装cnpm(taobao镜像)

3.全局安装gulp

3.1、说明：全局安装gulp目的是为了通过她执行gulp任务；
3.2、安装：命令提示符执行cnpm install gulp -g

4.新建package.json文件

4.1、说明：package.json是基于nodejs项目必不可少的配置文件，它是存放在项目根目录的普通json文件；
4.2、安装cnpm，执行命令cnpm init，可快速获取版本信息

5.本地安装gulp插件

5.1、安装：定位目录命令后提示符执行cnpm install --save-dev；
5.2、本示例以gulp-less为例（编译less文件），命令提示符执行cnpm install gulp-less --save-dev；

6.新建gulpfile.js文件（重要）

6.1、说明：gulpfile.js是gulp项目的配置文件，是位于项目根目录的普通js文件（其实将gulpfile.js放入其他文件夹下亦可）。 具体操作请到百度搜索（gulp安装）
项目结构 
项目结构介绍
├─app //html目录
│  │─components //组件模板文件夹（下面有多个组件文件夹）
│  │  ├─alert
│  │  ├─button
│  │  └─...
│  │─css //css文件夹（下面是sass文件解析导入的css文件）
│  │  ├─font //自定义icon图标文字库
│  │  ├─i //公用图标文件夹
│  │  └─tobe //组件样式文件
│  │─images //数据图片
│  │─js //js文件（下面有组件js与公用js）
│  │  ├─checkbox
│  │  ├─input
│  │  └─...
│  │─*.html //html文件
│  └─*.json //json文件
├─node_modules  //淘宝镜像文件
├─sass //sass文件（用来导出最终css文件，里面包含组件sass、公用与自定义sass文件）
│  │─font
│  └─tobe //组件sass文件
│─gulpfile.js //gulp配置文件
└─package.json //版本信息

app文件夹包含html主要文件。
node_modules文件夹是淘宝镜像文件，需要命令生成（node_modules包含多个插件）。
sass文件夹用来存放scss文件，scss通过命令解析导出最终css文件（app->css文件夹，sass中文件夹tobe为存放组件部分scss文件夹）。
components文件夹包含多个组件。
css文件夹为sass文件夹最终解析导出文件（css文件夹中tobe为组件css，i为icon图标文件夹，font为自定义icon图标文字库）。
images文件夹用来存放数据图片（会随数据的变更的图片，可删除图片）。
js文件夹用来存放页面逻辑或者效果处理的js文件（其中包括多个组件效果js文件）。
*.html，*.json文件为编写文件，需要添加页面在此处新建编写，其中*.html文件为html代码编写文件，*.json文件问数据文件（数据包含组件数据与页面基础数据，如引用的css文件、js文件或者html页面标题title）。
其中*.html、*.json需要在同一级文件、名称需要相同，否则无效。
gulpfile.js为gulp配置文件。
package.json为版本信息。

实例 
html代码结构
{{> head/head}}  //头部组件
<body>
    <div>
    //正文部分
    </div>
    {{> footer/footer}}  //尾部组件
</body>
</html>

{{> head/head}}头部组件，{{> footer/footer}}尾部组件

json数据结构
{
    "gl":{
        "title": "网页标题", //头部标题
        "styles": ["css/tab.css"], //页面样式
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],  //js库
        "require": "js/tab.js" //页面js
    }
}

title、styles头部组件内的参数（title标题、styles样式表）。scripts、require尾部组件参数（scripts，js库、require，自定义js）。
