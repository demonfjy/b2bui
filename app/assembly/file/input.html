{{type}}输入框类型。{{id}}定义按钮id。{{name}}输入框名称。{{class}}样式。{{#if disabled}}判断是否可点击。

组件样式调用（sass文件或css文件）
@import "icon";
@import "tobe/components/input/input";

这是组件所需调用的组件样式。

组件结构调用
{{#input1}}
    {{> input/input}}
{{/input1}}
{{#input2}}
    {{> input/input}}
{{/input2}}

{{#input1}}、{{#input2}}为json数据名称，可自行定义，{{> input/input}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/input.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/input.js"
    },
    "input1":{
        "type": "text",
        "id":"form-company", 
        "name": "form-txt", 
        "class" : "bl-input-lg",
        "placeholder": "请输入属性内容",
        "style": "bl-col-xs-12 bl-col-sm-12"
    },
    "input2":{
        "iconinput":"1",
        "id":"",
        "class":"",
        "placeholder":"请输入属性内容",
        "iconclass":"",
        "name":["1","2","3","4"]
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“input1”、“input2”组件调用数组名称一致。

js调用
require(['input/input'], function() {});

此处调用的是组件效果的js，如果不调用组件js不会产生效果。
