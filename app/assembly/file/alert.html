提示 alert

alert提示组件，组件提供了大致8种样式的提示组件。如有自定义组件可通过{{class}}修改。
项目结构
{{#alert1}} {{> alert/alert}} {{/alert1}} {{#alert2}} {{> alert/alert}} {{/alert2}} {{#alert3}} {{> alert/alert}} {{/alert3}} {{#alert4}} {{> alert/alert}} {{/alert4}} {{#alert5}} {{> alert/alert}} {{/alert5}} {{#alert6}} {{> alert/alert}} {{/alert6}} {{#alert7}} {{> alert/alert}} {{/alert7}} {{#alert8}} {{> alert/alert}} {{/alert8}} 
组件结构
<div class="bl-alert {{class}}">
    <i class="blicon {{icon}} bl-alert-icon"></i>
    <span class="bl-alert-message">{{name}}</span>
    <span class="bl-alert-description">{{message}}</span>
    <a class="bl-alert-close-icon">
        {{#if close.close1}}
            <i class="blicon blicon-error bl-alert-close"></i>
        {{/if}}
        {{#if close.close2}}
            <a class="bl-alert-close-icon">不再提醒</a>
        {{/if}}
    </a>
</div>

{{class}}此处为json数据，可通过json自行定义class。例如，我需要分页向右移动10像素，可通过此处类名修改。{{icon}}可用来更改图标样式。{{name}}需要提示的内容。{{message}}提示的详细信息。{{#if close.close1}}、{{#if close.close2}}判断显示关闭还是不再提醒

组件样式调用（sass文件或css文件）
@import "icon";//图标样式
@import "tobe/components/alert/alert";

这是组件所需调用的组件样式。

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“alert1”、“alert2”、“alert3”、“alert4”、与组件调用数组名称一致。class样式有四种，分别是bl-alert-error、bl-alert-success、bl-alert-warning、bl-alert-info，双行需要添加类bl-alert-with-description。icon样式有三种，分别是blicon-alert-error、blicon-alert-ok、blicon-alert-warning。
