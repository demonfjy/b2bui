上传 file

file 上传，组件主要用来上传图片。
项目结构
{{#file1}} {{> file/file}} {{/file1}} {{#file2}} {{> file/file}} {{/file2}} 
组件结构
<div class="bl-upload bl-upload-select bl-upload-select-picture-card {{class}}">
    <span tabindex="0" class="bl-upload" role="button">
        <input type="file" id="{{id}}" name="{{name}}" accept="" multiple title=" ">
        <div>
            <i class="blicon blicon-add"></i>
            <div class="bl-upload-text">{{text}}</div>
        </div>
    </span>
    <div class="bl-upload-img">
        {{#if img}}
            <img src="{{src}}">
        {{/if}}
    </div>
    <div class="bl-upload-x">X</div>
</div>

{{class}}自定义样式，{{name}}上传框名称，{{text}}上传框提示。{{#if img}}判断是否用默认图片。

组件样式调用（sass文件或css文件）
@import "icon";  //icon
@import "tobe/components/file/file";

这是组件所需调用的组件样式。

组件结构调用
{{#file1}}
    {{> file/file}}
{{/file1}}
{{#file2}}
    {{> file/file}}
{{/file2}}

{{#file1}}、{{#file2}}为json数据名称，可自行定义。{{> file/file}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/file.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/file.js"
    },
    "file1": {
        "class": "",
        "id": "",
        "name": "",
        "text": "相关附件"
    },
    "file2": {
        "class": "",
        "id": "",
        "name": "",
        "text": "相关附件",
        "img": "1",
        "src": "1.jpg"
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“file1”、"file2"与组件调用数组名称一致。

js调用
require(['file/file'], function() {});

此处调用的是组件效果的js，如果不调用组件js不会产生效果。
