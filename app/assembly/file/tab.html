{{tabtype}}此处为json数据，可通过json自行定义class。例如，我需要分页向右移动10像素，可通过此处类名修改。{{id}}定义按钮id。{{#if disabled}}判断时候可点击。

组件样式调用（sass文件或css文件）
@import "icon";
@import "tobe/components/tab/tab";

这是组件所需调用的组件样式。

组件结构调用
{{#tab1}}
    {{> tab/tab}}
{{/tab1}}

{{#tab1}}为json数据名称，可自行定义，{{> tab/tab}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/tab.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/tab.js"
    },
    "tab1" : {
        "tabtype" : "aaa",
        "tabname" : "aaa",
        "name" : [{
            "active" : ""
        },{
            "active" : "1"
        },{
            "active" : ""
        }]
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“tab1”组件调用数组名称一致。
