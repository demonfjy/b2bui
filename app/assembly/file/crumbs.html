面包屑 crumbs

checkbox 面包屑，一般用来放置页面路径或者页面位置。
项目结构
{{#crumbs1}} {{> crumbs/crumbs}} {{/crumbs1}} 
组件结构
<div class="bl-crumbs">
    {{#each title}}
        <span class="bl-crumbs-title"><a href="" title="{{this}}">{{this}}</a></span>
    {{/each}}
    {{#each name}}
        <span class="bl-crumbs-name"><a href="" title="{{this}}">{{this}}</a></span>
    {{/each}}
</div>

title、name路径与页面名称。

组件样式调用（sass文件或css文件）
@import "tobe/components/crumbs/crumbs";

这是组件所需调用的组件样式。

组件结构调用
{{#crumbs1}}
    {{> crumbs/crumbs}}
{{/crumbs1}}

{{#crumbs1}}为json数据名称，可自行定义。{{> crumbs/crumbs}}为组件在项目中的位置。

json数据结构
{
    "gl":{
        "title": "网页标题",
        "styles": ["css/crumbs.css"],
        "scripts": ["http://k10.iblimg.com/resources/lib/jquery-1.8.2.js"],
        "require": "js/crumbs.js"
    },
    "crumbs1":{
        "title": ["用户注册","用户注册"], 
        "name": ["用户注册","用户注册"]
    }
}

“style”值为组件样式表名称，如页面有多个不同组件，此处可继承多个组件样式表。例如：页面有按钮组件，此处为"style" : ["page","button","..."]。“script”为js组件库，或其他js。“require”为此页面 js（自行编写）。“crumbs1”与组件调用数组名称一致。
